# Document Q&A System

## 📌 Overview
This project allows users to ask questions based on a document. The system retrieves the most relevant answer using NLP techniques.

## 🚀 Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## ⚙️ Features
- Text processing
- Query matching
- Similarity-based answer retrieval

## 🧠 How it Works
The document is converted into vectors using TF-IDF. User queries are compared using cosine similarity to find the most relevant answer.

## ▶️ Run the Project
pip install -r requirements.txt  
python main.py
